# Latticework: Invert

One-shot. Take the decision or plan in question and reason backward.

1. **Assume it failed.** Project six to twelve months out and state plainly: it
   did not work. Write the one-sentence obituary.
2. **List the killers.** Name the three to five most likely causes of that
   failure, concrete and specific to this situation, not generic risks.
3. **Rank by likelihood x damage.** Mark which are probable, which are fatal,
   which are both.
4. **Design the fix.** For each top killer, name the cheapest thing that removes
   or de-risks it now.
5. **Verdict.** Is the plan worth doing once the top killers are addressed, or
   does inversion reveal it should not proceed?

Keep it concrete. The value is in naming the specific ways this loses, not in
reciting that "things can go wrong."
